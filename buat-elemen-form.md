# Buat Elemen Form

Contoh :

Anda dapat membuat form web yang benar-benar mengirimkan data ke server menggunakan tidak lebih dari HTML murni. Anda dapat melakukan ini dengan menentukan tindakan pada elemen formu Anda.

Sebagai contoh :

```
<form action = "/ url-where-you-want-to-submit-form-data"> </ form>
```

Sarang bidang teks Anda dalam elemen formulir. Tambahkan atribut action = "/ submit-cat-photo" ke elemen formulir ini.

Soal :

1. Sarang elemen input teks Anda dalam elemen form 
2. Pastikan form Anda memiliki atribut aksi yang disetel ke / kirim-cat-foto 
3. Pastikan elemen form Anda memiliki tag penutup.

```css
<link href="https://fonts.googleapis.com/css?family=Lobster" rel="stylesheet" type="text/css">
<style>
  .red-text {
    color: red;
  }

  h2 {
    font-family: Lobster, Monospace;
  }

  p {
    font-size: 16px;
    font-family: Monospace;
  }

  .thick-green-border {
    border-color: green;
    border-width: 10px;
    border-style: solid;
    border-radius: 50%;
  }

  .smaller-image {
    width: 100px;
  }
</style>

<h2 class="red-text">CatPhotoApp</h2>

<p>Click here for <a href="#">cat photos</a>.</p>

<a href="#"><img class="smaller-image thick-green-border" alt="A cute orange cat lying on its back. " src="https://bit.ly/fcc-relaxing-cat"></a>

<p>Things cats love:</p>
<ul>
  <li>cat nip</li>
  <li>laser pointers</li>
  <li>lasagna</li>
</ul>
<p>Top 3 things cats hate:</p>
<ol>
  <li>flea treatment</li>
  <li>thunder</li>
  <li>other cats</li>
</ol>
<input type="text" placeholder="cat photo URL">
```
