# Tambahkan Teks Placeholder ke Bidang Teks

Teks placeholder Anda adalah apa yang muncul dalam input teks Anda sebelum pengguna Anda memasukkan apa pun.

Anda dapat membuat teks placeholder seperti:

```
<input type = "text" placeholder = "ini teks placeholder">
```

Setel nilai placeholder dari input teks Anda ke "cat photo URL".

Soal :

1. Tambahkan elemen input teks atribut placeholder.
2. Setel nilai atribut placeholder Anda ke "cat photo URL".
3. Elemen masukan yang sudah jadi harus memiliki sintaks yang valid.

```
<link href="https://fonts.googleapis.com/css?family=Lobster" rel="stylesheet" type="text/css">
<style>
  .red-text {
    color: red;
  }

  h2 {
    font-family: Lobster, Monospace;
  }

  p {
    font-size: 16px;
    font-family: Monospace;
  }

  .thick-green-border {
    border-color: green;
    border-width: 10px;
    border-style: solid;
    border-radius: 50%;
  }

  .smaller-image {
    width: 100px;
  }
</style>

<h2 class="red-text">CatPhotoApp</h2>

<p>Click here for <a href="#">cat photos</a>.</p>

<a href="#"><img class="smaller-image thick-green-border" alt="A cute orange cat lying on its back. " src="https://bit.ly/fcc-relaxing-cat"></a>

<p>Things cats love:</p>
<ul>
  <li>cat nip</li>
  <li>laser pointers</li>
  <li>lasagna</li>
</ul>
<p>Top 3 things cats hate:</p>
<ol>
  <li>flea treatment</li>
  <li>thunder</li>
  <li>other cats</li>
</ol>

<input type="text">
```
